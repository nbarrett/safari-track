# Klaserie Camps

Tracking game drives and wildlife sightings at Klaserie Private Nature Reserve, South Africa.

## Structure

```
drives/          Game drive logs organised by date
  YYYY-MM-DD-am.md   Morning drive
  YYYY-MM-DD-pm.md   Afternoon/evening drive
sightings/       Photo index and notable sighting records
maps/            Drive route maps and area references
```

## Drive Log Format

Each drive file records:
- Date, time (departure/return), and session (AM/PM)
- Guide and tracker names
- Vehicle number
- Weather conditions
- Route summary
- Sightings with timestamps, species, count, location, and behaviour
- Highlights and notes
